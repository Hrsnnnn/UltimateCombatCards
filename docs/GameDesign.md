## 二、世界观与阵营

### 核心对立
**人类（Human）** vs **非人类（Non-Human）**

- **人类方**：血肉之躯，依靠肉体极限、智谋、财富、精神信念、超自然能力战斗
- **非人类方**：超自然存在，神明、亡灵、怪兽、机械生命、异星生物

> 角色来源：动漫、历史人物、电影IP、体育明星、神话传说等跨IP混搭。阵营划分以"是否为血肉之躯的人类"为准，而非善恶。

## 五、超能力卡系统

### 基本规则
- 每位英雄拥有 **4 张**专属超能力卡（同一张牌的4份）
- **费用：统一为1**
- **使用：用完即消耗**（每张独立计算）
- 游戏开始时，**1 张**进入初始手牌，剩余 **3 张**置入英雄专属的超能力池
- 超能力不受法免（Untrickable）限制

### 护盾系统（Shield）

每位英雄拥有一条**护盾条**，共 **8 格**，用于积累格挡能量。

#### 充能规则
- 每次英雄**被攻击时**，护盾条随机增长 **1～3 格**
- 护盾条积累满 **8 格**时，触发**格挡**

#### 格挡触发效果
1. **免疫本次伤害**：该次攻击对英雄造成的伤害被完全抵消
2. **护盾条清零**：8 格归零，重新开始积累
3. **从超能力池抽取 1 张**超能力牌，玩家可选择：
   - **立即打出**：不消耗任何费用，当场生效
   - **加入手牌**：拿到手中，后续回合按正常方式打出（消耗 1 费）

#### 超能力池耗尽
- 超能力池中的 3 张牌全部用完后，护盾条**停止积累**，无法再触发格挡
- 每局英雄最多触发 **3 次**格挡

## 六、回合结构

### 胜负条件
- **对方英雄 HP 降至 0** 即获胜
- **对方牌库耗尽**（无牌可抽时）即获胜

### 费用规则
- 双方初始费用均为 **1**，每回合开始自动 **+1**（无上限）
- 未使用的费用**不跨回合累积**，每回合重置

### 四阶段回合流程

每回合按顺序经历四个阶段：

| 阶段 | 行动方 | 可做操作 |
|------|--------|----------|
| 第一阶段：非人类方出牌 | 非人类英雄 | 打出非人类单位 |
| 第二阶段：人类方出牌 | 人类英雄 | 打出人类单位、法术牌、环境牌 |
| 第三阶段：非人类方法术 | 非人类英雄 | 打出法术牌、环境牌（特殊情况下可打出单位）|
| 第四阶段：战斗！ | 双方 | 场上所有单位从左至右依次进攻，**非人类方单位先攻** |

### 战斗规则
- 同一条道路上的单位互相攻击
- 若己方道路上没有敌方单位，则直接攻击对方英雄
- **每张单位每回合只参与一次攻击结算**（除非拥有双击、狂暴等额外攻击关键词）
- **存活的单位留场**：战斗结算后未被消灭的单位继续留在道路上，进入下一回合
- 每回合开始时自动为当前玩家补充一张手牌

## 七、卡牌系统

### 卡牌类型
| 类型 | 说明 |
|------|------|
| 单位（Unit）| 放置在道路上，有攻击力/生命值，可攻击 |
| 法术（Spell）| 即时效果，使用后消耗 |
| 环境（Environment）| 放置在道路上，持续效果，每条道路只能有一张 |

### 稀有度（5档）
| 稀有度 | 说明 |
|--------|------|
| 普通 | 基础单位和法术，费用低，效果简单 |
| 非普通 | 有特殊效果的中低费单位 |
| 稀有 | 有复杂效果的中费单位 |
| 超稀有 | 强力高费单位，具有关键词组合 |
| 传奇 | 每副卡组最多 4 张，独特效果 |

### 手牌上限
- 手牌上限为 **10 张**
- 超过上限时：无法抽牌、无法召唤、无法触发护盾格挡

### 卡组构建规则
- 每副卡组 **40 张**
- 只能使用英雄所属2个属性下的卡牌
- 同一张普通/非普通/稀有/超稀有/**传奇**卡最多放 **4 张**

### 初始手牌
- 游戏开始时手牌共 **5 张**：
  - 4 张从卡组中随机抽取
  - 1 张为英雄专属超能力牌（置于专属栏位，不计入卡组）

### 道路系统

游戏共有 **5 条道路**，从左到右：

| 位置 | 类型 | 说明 |
|------|------|------|
| 最左（第1条）| 高地面 | 特殊地形，待细化 |
| 第2、3、4条 | 平地 | 普通道路 |
| 最右（第5条）| 水路 | 仅具有「两栖」关键词的卡牌可放置 |

> ⚠️ 高地面的具体规则效果待后续讨论确认。

## 七、关键词系统（24个）

| 关键词 | 效果描述 |
|--------|---------|
| 协作（Team-Up）| 可与其他带协作的单位共占同一格 |
| 两栖（Amphibious）| 可放置在水域道路 |
| 必中（Bullseye）| 攻击英雄时不填补对方护盾条 |
| 神射（Foresight X）| 战斗阶段开始前，先对敌方英雄造成 X 点伤害 |
| 致命（Deadly）| 造成任意伤害即消灭目标 |
| 护甲X（Armored X）| 受到伤害减少X点（护甲用尽后正常受伤）|
| 穿透（Strikethrough）| 攻击时对同道路上所有单位依次造成伤害，剩余攻击力最终打到英雄 |
| 溢伤X（Overshoot X）| 消灭对面单位后，对该道路后方的下一目标（有单位先打单位，否则打英雄）造成固定 X 点伤害 |
| 双击（Double Strike）| 战斗后可再攻击一次 |
| 克主X（Anti-Hero X）| 对方该道路无单位时，攻击力额外+X |
| 追踪（Hunt）| 出场时立刻移动到有敌方单位的道路 |
| 法免（Untrickable）| 不受法术卡影响（英雄超能力不受此限制）|
| 潜伏（Gravestone）| 以隐藏状态放置，对方不可见，战斗前揭示 |
| 揭示（Reveal）| 从潜伏状态显形时触发的额外效果 |
| 狂暴（Frenzy）| 消灭一个单位后若自身存活，立即再攻击一次（**非人类方专有**）|
| 溅射X（Splash X）| 攻击时对相邻道路各造成X点伤害 |
| 冻结（Freeze）| 该单位本回合无法攻击 |
| 击退（Knockback）| 将目标单位退回其持有者手牌，可再次打出 |
| 感知（Sense）| 玩家抽到此牌时，立即触发一次该单位的能力 |
| 进化（Evolution）| 可叠加在指定单位上打出，触发额外增益效果 |
| 血战（Bloodrush）| 以当前生命值代替攻击力进行战斗 |
| 无敌（Invincible）| 无法受到伤害（即消灭类效果仍然有效）|
| 迟钝（Blunted）| 该单位无法触发双击或狂暴 |
| 召唤（Conjure）| 从符合条件（指定族群或属性）的**全卡池**中随机选取一张牌加入手牌（非当前卡组）|
