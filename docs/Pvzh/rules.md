## 游戏模式

| 模式 | 说明 |
|------|------|
| 植物任务 | 玩家扮演植物，对抗僵尸 |
| 僵尸任务 | 玩家扮演僵尸，对抗植物 |
| 多人游戏 | 可扮演植物或僵尸，与全球真实玩家对决 |
| 今日挑战 | 每天参与官方设计的挑战活动，赢取活动票券和免费组合包 |

## 游戏玩法

- **主要目标**：将对方英雄生命值降至 0（或对方牌库耗尽）即可获胜。
- 通过在战场上出牌来与对方进行战斗。

### 卡牌分类

| 类型 | 说明 |
|------|------|
| 队友牌 | 植物方称"植物"，僵尸方称"僵尸"；留在战场上直到被消灭 |
| 锦囊牌 | 打出后立即产生效果，不留场 |
| 环境牌 | 布置在某条线上，持续发挥效果 |

### 队友属性

每个队友拥有三项基础属性：
- **攻击力**：对敌人造成伤害的数值
- **生命值**：能承受伤害的上限
- **费用**：植物消耗阳光，僵尸消耗脑子

## 回合结构

每回合分为四个阶段，按顺序进行：

| 阶段 | 行动方 | 可做操作 |
|------|--------|----------|
| 第一阶段：僵尸出牌 | 僵尸英雄 | 打出僵尸队友 |
| 第二阶段：植物出牌 | 植物英雄 | 打出植物队友、植物锦囊牌、植物环境牌 |
| 第三阶段：僵尸锦囊牌 | 僵尸英雄 | 打出僵尸锦囊牌、僵尸环境牌（特殊情况下可打出僵尸队友） |
| 第四阶段：开战！ | 双方 | 场上所有队友从左至右依次进攻，僵尸永远先攻击 |

### 战斗规则

- 同一条线上的队友互相攻击。
- 若自己线上没有敌方队友，则对方英雄直接承受伤害。
- 每回合开始，阳光/脑子的上限 **+1**，玩家可打出更高费用的牌。

## 英雄系统

- 每个英雄拥有 **20 点生命值**。
- 每个英雄具备 **两个属性**，决定其可使用的卡牌范围。
- 每个英雄拥有 **四张超能力**（不计入牌组）：
  - 一张**专属超能力**（能力尤为强大）
  - 三张**属性超能力**（由英雄属性决定）

## 超能格挡与超能力

- 英雄承受伤害时，超能格挡槽**随机填补 1-3 格**。
- 当格挡槽达到 **8 格或以上**时，英雄获得一张超能力（可立即免费打出，或保留备用）。
- 每场游戏中英雄可以格挡 **最多 3 次**，每次格挡后小盾牌减少一个。
- 第三次格挡后，格挡槽破碎，英雄**再也无法格挡**。
- 游戏开始时会先提供一张超能力。

## 卡牌类型

### 队友牌

- 有攻击力、生命值和费用三项属性。
- 留在战场上直到被消灭。
- 部分队友拥有特殊特性或能力。

### 锦囊牌

- 打出后立即生效，不留场。
- 效果包括：攻击/治疗、召唤/消灭队友、提高/降低属性等。
- 植物锦囊牌在**第二阶段**打出，僵尸锦囊牌在**第三阶段**打出。

### 环境牌

- 布置在某条线上，持续影响该线上的战斗。

### 超能力

- 费用为 1。
- 极为强大，不计入牌组。

---

## 牌组构建规则

- 每个牌组恰好 **40 张牌**（队友 + 锦囊牌 + 环境牌）。
- 另附四张超能力，**不计入**牌组数量。
- 同一张牌在牌组中最多 **4 张**。
- 手牌上限为 **10 张**；超过上限时无法抽牌、召唤或触发超能格挡。
- 卡牌规格分类：普通、高级、巨大、三叠纪。
- 卡牌稀有度：常见 → 罕见 → 稀有 → 超稀有 → 传说。

## 特性与效果

### 特性

| 特性 | 说明 | 阵营 |
|------|------|------|
| 两栖 | 可被打在任意一条线（包括水池线） | 双方 |
| 克制英雄 X | 自己线上没有敌方队友时，攻击力 +X | 双方 |
| 装甲 X | 所承受的伤害减少 X 点 | 双方 |
| 必中 | 攻击对方英雄时不填补其超能格挡槽 | 双方 |
| 致命 | 对队友造成任意伤害时直接消灭它 | 仅僵尸 |
| 双重打击 | 同线战斗后未被消灭，可额外攻击一次 | 仅植物 |
| 狂热 | 消灭一株植物后未被消灭，可额外攻击一次 | 仅僵尸 |
| 墓碑 | 在锦囊牌阶段开始前躲在墓碑里，不受大多数效果影响 | 仅僵尸 |
| 狩猎 | 当一名队友被打在某条线时，自动移动到该线 | 双方 |
| 先攻 X | 同线战斗前，先对敌方英雄造成 X 点伤害 | 仅僵尸 |
| 溅射伤害 X | 进攻时同时对相邻线造成 X 点伤害 | 仅植物 |
| 穿透 | 对同线所有队友造成伤害，包括对方英雄 | 双方 |
| 组队 | 该队友前后可再放置另一名队友 | 仅植物 |
| 锦囊免疫 | 不受任何锦囊牌影响（自己的除外） | 双方 |

### 效果

| 效果 | 说明 |
|------|------|
| 弹射 | 将一名队友退回持有者手牌，可再次打出 |
| 召唤 | 随机获得一张特定类型的牌 |
| 龙吼 | 玩家抽牌时，该队友触发其能力 |
| 进化 | 在特定队友上打出此牌以施展额外能力 |
| 冻结 | 使一名队友跳过一次攻击（触发额外攻击可解除） |
| 治疗 | 恢复队友或英雄的生命值 |
| 生命攻击 | 该队友以生命值作为攻击力计算 |
| 无法受到伤害 | 对手无法对该队友造成伤害（秒杀类锦囊牌仍可消灭） |
| 无法进行额外攻击 | 无法触发狂热或双重打击 |
