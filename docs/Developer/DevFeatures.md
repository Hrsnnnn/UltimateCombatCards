## 二、回合系统

### 2.1 回合流程（4阶段）

```
阶段1：非人类方出单位
  └─ 非人类方可打出：单位卡、环境卡
  └─ 超能力在任意己方阶段均可使用

阶段2：人类方出单位+法术
  └─ 人类方可打出：单位卡、法术卡、环境卡、超能力

阶段3：非人类方出法术
  └─ 非人类方可打出：法术卡、环境卡
  └─ 特殊情况下可出单位（特定卡牌效果触发）

阶段4：战斗阶段（自动结算）
  └─ 不接受玩家输入
  └─ 按道路0→4依次结算
  └─ 非人类方单位先攻

回合结束处理
  └─ 清除本回合临时效果
  └─ 双方各抽1张牌
  └─ 费用上限各+1（无上限）
  └─ 触发"回合结束"时机效果
```

### 2.2 费用系统

- [ ] 每回合开始自动回复至上限（不累积）
- [ ] 费用永久增加（商业帝国等，叠加计入上限）
- [ ] 本回合临时额外费用（IPO敲钟、财富裂变等）
- [ ] 费用翻倍（本回合，有最大值上限）
- [ ] 减少对方费用（资本运作、垄断市场等）
- [ ] 将剩余未使用费用转化为其他效果（垄断协议超能力）
- [ ] 费用上限临时减少（某些控制效果）

### 2.3 战斗结算顺序（单条道路内）

```
0. [战斗阶段开始前] 所有有神射关键词的单位触发预伤害（见神射实现要点）
1. 揭示该道路所有潜伏单位（触发揭示效果）
2. 非人类方单位攻击（每个单位本回合只攻击一次）
   a. 对面有单位 → 双方同时互相造成攻击力伤害（各受对方当前攻击力伤害）
   b. 对面无单位 → 直接攻击人类英雄
3. 死亡检测（生命值≤0 → 消灭 → 触发消灭效果 → 移入墓地）
4. 人类方单位攻击（同上逻辑）
5. 再次死亡检测
6. 检测游戏是否结束
```

> ⚠️ **关键规则：每个单位每次战斗只攻击一次**
> - 攻击者击杀了防守单位后，**不会**继续将剩余伤害传递给对方英雄
> - 即：「击杀换取」——击杀对面单位，战斗在该道路结束，英雄不额外受伤
> - **唯一例外**：拥有穿透（PIERCE）或溢伤（OVERSHOOT）关键词的单位在击杀后可对英雄造成额外伤害（但仍是该次攻击的一部分，不是额外攻击）
> - 双击/狂暴等关键词允许发起额外攻击，但每次额外攻击仍遵循上述规则

## 三、道路与战场系统

### 3.1 道路类型

| 道路编号 | 类型 | 特殊规则 |
|----------|------|---------|
| 道路0（最左）| **高地面** | 无特殊放置规则；部分卡牌在高地打出时触发额外效果 |
| 道路1 | 平地 | 无 |
| 道路2 | 平地 | 无 |
| 道路3 | 平地 | 无 |
| 道路4（最右）| **水路** | 仅拥有「两栖」关键词的单位可放置 |

> ⚠️ 高地面道路的具体特殊规则待设计方确认

### 3.2 道路容量规则

- [ ] 默认每条道路每方最多 **1个单位**
- [ ] 拥有「协作」关键词的单位可与其他协作单位共占同一道路
- [ ] 协作单位之间不互相阻挡，均参与战斗
- [ ] 协作上限：同道路同方最多 **2个协作单位**
- [ ] 协作单位攻击顺序：**距离对方英雄越近的越先攻击**（即靠近对方区域的单位优先）
- [ ] 环境牌：每条道路每方仅能有 **1张**，新环境牌替换旧的

## 四、卡组与手牌系统

- [ ] 卡组构建：选择英雄后，仅能使用该英雄两个属性下的卡牌
- [ ] 卡组容量：恰好40张
- [ ] 单卡数量：传奇无限制，其余稀有度上限待确认
- [ ] 牌库洗牌（开局随机）
- [ ] 初始手牌：从牌库抽4张 + 超能力置入专属栏
- [ ] 每回合开始自动抽1张
- [ ] 主动抽牌（兵法三十六计、垄断协议等效果触发）
- [ ] 手牌上限10张，超出时溢出的牌直接丢弃
- [ ] 弃牌机制（主动弃牌、被效果强制弃牌）
- [ ] 牌库耗尽判定（胜负条件之一）
- [ ] 从牌库搜索特定条件的卡牌（钢铁侠超能力等）
- [ ] 将手牌放回牌库（诸葛亮超能力）
- [ ] 查看对方手牌（蝙蝠侠超能力、黄蓉等）
- [ ] 查看对方牌库顶（侦察无人机等）
- [ ] 封锁对方手牌中某张牌本回合无法打出（蝙蝠侠超能力）

## 五、英雄系统

- [ ] 英雄初始血量（待确认，统一实现）
- [ ] 英雄受伤（被直接攻击 或 效果直接伤害）
- [ ] 英雄治疗（不超过初始血量上限）
- [ ] 英雄血量归零 → 游戏结束
- [ ] 英雄拥有2个属性标签（决定可用卡牌范围）
- [ ] 超能力专属栏：游戏开始时置入，独立于手牌
- [ ] 超能力：1费，一次性，使用后消耗（不进入墓地）
- [ ] 超能力需要目标时走正常目标选择流程
- [ ] 英雄在场外，不占道路，不参与战斗结算（仅承受直接伤害）

## 六、卡牌类型实现

### 6.1 单位卡

- [ ] 放置到指定道路（合法性检验：费用/道路类型/协作规则）
- [ ] 攻击力 / 生命值 / 最大生命值管理
- [ ] 登场触发效果（On Play）
- [ ] 消灭触发效果（On Death）
- [ ] 受伤触发效果（On Damaged）
- [ ] 持续效果（Ongoing，一直生效直到单位离场）
- [ ] 临时效果（本回合结束后清除）
- [ ] 永久增益效果（本局游戏持续）

### 6.2 法术卡

- [ ] 即时生效，使用后进入弃牌堆
- [ ] 有目标法术：需玩家选择合法目标
- [ ] 无目标法术：直接确认使用
- [ ] 「法免」单位不能被**法术卡**或**超能力卡**指定为目标；单位自身效果可正常影响法免单位
- [ ] 无法被反制标记（部分超能力效果）

### 6.3 环境卡

- [ ] 放置到指定道路
- [ ] 每条道路每方1张环境，新的替换旧的（旧环境消失，不触发消灭效果）
- [ ] 环境持续效果（每回合触发 或 条件触发）
- [ ] 环境被替换或被效果消灭时的处理

### 6.4 超能力卡

- [ ] 独立于手牌的专属槽位
- [ ] 与普通卡牌相同的出牌流程（消耗1费）
- [ ] 使用后槽位清空（全局仅一次）

## 七、关键词系统（23个）

每个关键词独立实现，可动态附加/移除。

> 关键词定义以 `GameDesign.md` §七为权威来源，`CardsCommon.md` 为分类索引。
> 本节只记录**实现要点**，不重复规则描述。

### 位置与移动（3个）

| 关键词 | 实现要点 |
|--------|---------|
| **协作（Team-Up）** | 允许多个协作单位共占同一道路；最多2个；攻击顺序按距对方英雄由近到远 |
| **两栖（Amphibious）** | 无此关键词的单位无法放置到道路4（水路）|
| **追踪（Hunt）** | 出场后自动寻找有敌方单位的道路；多条时随机选一条 |

### 攻击与伤害（9个）

| 关键词 | 实现要点 |
|--------|---------|
| **必中（Bullseye）** | 对英雄造成伤害时跳过护盾充能逻辑 |
| **神射（Foresight X）** | 在战斗结算循环启动**之前**触发，造成固定X点伤害；X为卡牌上标注的数值 |
| **穿透（Strikethrough）** | 击杀对面单位后，`overkill = 攻击力 − 目标剩余生命值`，对英雄造成overkill伤害 |
| **溢伤X（Overshoot X）** | 击杀对面单位后，对该道路后方下一目标（有单位则打单位，否则打英雄）造成固定X点伤害 |
| **双击（Double Strike）** | 本回合战斗结束后若自身存活，立刻对同道路再攻击一次；需标记防止无限循环 |
| **克主X（Anti-Hero X）** | 攻击前检测对面道路是否有单位，无单位则临时攻击力+X |
| **狂暴（Frenzy）** | 击杀后若存活，对下一目标（同道路或英雄）再攻击一次 |
| **溅射X（Splash X）** | 攻击时同步对左右相邻道路的单位（无单位则英雄）造成X点伤害 |
| **血战（Bloodrush）** | 攻击时用 `currentHealth` 替换 `currentAttack` 参与伤害计算 |

### 防御（2个）

| 关键词 | 实现要点 |
|--------|---------|
| **护甲X（Armored X）** | 受伤时先消耗护甲值，超出部分才扣生命值；护甲可叠加 |
| **无敌（Invincible）** | 受到任何来源的数值伤害时归零处理；消灭/移除/弹回效果正常生效 |

### 特殊能力（7个）

| 关键词 | 实现要点 |
|--------|---------|
| **致命（Deadly）** | 对单位造成≥1点伤害时，目标直接进入死亡流程（无视剩余生命值）|
| **法免（Untrickable）** | 法术卡无法以法免单位为目标；超能力卡**不受此限制**，可正常指定法免单位 |
| **潜伏（Gravestone）** | 放置时 `isAmbushed=true`；对方无法看到该单位；战斗阶段开始前统一揭示 |
| **揭示（Reveal）** | 揭示时机独立于登场（ON_PLAY），触发 ON_REVEAL 效果 |
| **感知（Sense）** | 摸牌时检测，若摸到的牌有感知关键词，立即触发其能力一次（不需要打出）|
| **进化（Evolution）** | 打出时可选择一个己方已有单位作为宿主，叠加后触发进化效果；进化牌不独立占道路 |
| **召唤（Conjure）** | 从符合条件的全卡池（非当前卡组）中随机抽取一张牌加入手牌；需独立卡池数据源 |

### 控制与位移（3个）

| 关键词 | 实现要点 |
|--------|---------|
| **冻结（Freeze）** | 设置 `isFrozen=true`；战斗结算时跳过冻结单位的攻击；回合结束后清除 |
| **击退（Knockback）** | 将目标单位移出战场，放回其主人手牌（重置为原始状态）；可再次打出 |
| **迟钝（Blunted）** | 战斗时检测，若单位有迟钝标记则跳过双击/狂暴的额外攻击逻辑 |

## 八、效果类型系统

### 8.1 伤害效果

- [ ] 单体伤害（指定单位/英雄）
- [ ] 全体伤害（所有敌方/所有单位不区分敌我）
- [ ] 随机伤害（随机N个目标）
- [ ] 条件最高/最低伤害（生命值最低的单位等）
- [ ] 无视护甲的伤害（金刚狼超能力）
- [ ] 不可反制的伤害（部分超能力）
- [ ] 毒素伤害（每回合-N生命，持续M回合）

### 8.2 治疗效果

- [ ] 英雄治疗（不超过初始血量上限）
- [ ] 单位治疗（不超过最大生命值）
- [ ] 全体单位治疗（千手柱间等）
- [ ] 吸血（造成伤害时为英雄治疗等量，德古拉等）
- [ ] 治疗至满血（复活/千手柱间回合开始）
- [ ] 禁止治疗状态（永恒诅咒，目标无法被治疗）

### 8.3 强化/弱化效果

- [ ] 攻击力/生命值 +N/-N（数值型）
- [ ] 攻击力翻倍（本回合 / 永久）
- [ ] 攻击力=生命值（花木兰·沙场征袍）
- [ ] 攻击力=场上友方单位数×2（孙悟空龙珠）
- [ ] 最大生命值修改（扩大时同步增加当前生命）
- [ ] 关键词附加/移除（动态增加或取消关键词）
- [ ] 护甲破坏（清除所有护甲值，牛魔王/电磁脉冲）
- [ ] 免疫削弱效果（任正非，友方单位不受-攻击/-生命影响）
- [ ] 攻击力强制变为1/0（现实扭曲/达克赛德，本回合）

### 8.4 召唤效果

- [ ] 随机召唤（随机属性/族群/费用范围）
- [ ] 指定召唤（固定单位，如蒙古骑兵）
- [ ] 在指定道路召唤
- [ ] 在随机道路召唤
- [ ] 在所有道路各召唤一个
- [ ] 从牌库搜索召唤（钢铁侠超能力）
- [ ] 免费打出牌库中的牌（乔布斯超能力）
- [ ] 召唤代币单位（代币不进入牌库）

### 8.5 弹回效果

- [ ] 将单位送回其主人手牌（可重新打出）
- [ ] 弹回并增加目标下次费用（苏格拉底）
- [ ] 弹回对方所有单位（孙子·兵法真传）

### 8.6 消灭效果

- [ ] 直接消灭指定单位（霸王举鼎等）
- [ ] 消灭生命值≤N的单位（终结者超能力）
- [ ] 消灭攻击力≤N的单位（赫拉克勒斯·十二伟业等）
- [ ] 消灭生命值最低/最高的单位
- [ ] 消灭攻击力最高的单位
- [ ] 消灭费用最高的单位（弹射陷阱）
- [ ] 随机消灭场上一半单位（灭霸·无限手套）
- [ ] 消灭不触发任何消灭效果（宇宙黑洞/霍金·黑洞理论）
- [ ] 消灭且无法被复活（宇宙黑洞）
- [ ] 消灭所有单位（末日大决战类）
- [ ] 计时消灭（3回合后必定消灭，阎王·生死簿）

### 8.7 移动效果

- [ ] 将己方单位移动到指定道路（赫尔墨斯）
- [ ] 将对方单位移动到另一条道路（围魏救赵）
- [ ] 交换两个单位位置（罗·手术果实）
- [ ] 将对方所有单位随机重新分配（次元穿梭）
- [ ] 单位自动追踪移动（追踪关键词）
- [ ] 单位被传送并做额外攻击（量子跃迁）

### 8.8 复制/变形效果

- [ ] 复制己方单位到另一条道路（鸣人超能力）
- [ ] 复制对方单位加入己方（洛基·幻象）
- [ ] 将单位变为1/1（七十二变，保留关键词）
- [ ] 将单位变为随机等费单位（洛基·诡计）
- [ ] 将单位替换为另一个单位（奥丁弃牌召唤）
- [ ] 复制对方上一个打出的技能效果（卡卡西/宇智波佐助）
- [ ] 种族转换（奥创：将单位种族变为机甲）
- [ ] 升级普通单位为稀有（奥创·神级意识）

### 8.9 控制效果

- [ ] 永久夺取对方单位控制权（X教授超能力，费用≤3）
- [ ] 强制控制对方单位攻击一次（挟天子以令诸侯）
- [ ] 封锁对方手牌中某张牌本回合无法打出（蝙蝠侠超能力）
- [ ] 绑定两个单位（月老红线，一方受伤另一方同等受伤）

### 8.10 抽牌/搜索效果

- [ ] 抽N张牌
- [ ] 抽牌至手牌上限（触发弃牌）
- [ ] 从牌库搜索满足条件的牌加入手牌
- [ ] 将手牌中N张放回牌库顶（顺序自定，诸葛亮超能力）
- [ ] 从牌库顶查看N张（侦察无人机）

### 8.11 复活效果

- [ ] 从墓地复活指定单位（满血状态）
- [ ] 从墓地复活为1/1状态
- [ ] 批量复活费用≤N的墓地单位（哈迪斯超能力）
- [ ] 下回合自动复活（凤凰/杰森·永不消亡，每局限一次）
- [ ] 环境触发复活（轮回之地，随机复活）
- [ ] 被消灭标记为"不可复活"（宇宙黑洞）

### 8.12 特殊控制效果

- [ ] 冻结（目标跳过攻击，N回合）
- [ ] 跳过行动（宇智波斑·无限月读，跳过N回合攻击）
- [ ] 跳过对方完整回合（时间悖论）
- [ ] 对方下次法术效果随机变化（宇宙噪音）
- [ ] 无法进行额外攻击（特定状态）
- [ ] 本回合无法被消灭（无敌之躯，仍可受伤）
- [ ] 每回合只能被攻击一次（拉达冈·熔岩巨人）
- [ ] 单次格挡（神奇女侠，下次被攻击时不受伤，消耗后消失）

## 九、触发时机系统

所有触发时机需要独立的事件队列，按顺序处理，防止循环触发。

| 时机 | 说明 | 示例 |
|------|------|------|
| **登场时（ON_PLAY）** | 单位放置到战场 | 赫尔墨斯、宙斯 |
| **揭示时（ON_REVEAL）** | 潜伏单位翻转 | 杰森、掠食者 |
| **消灭时（ON_DEATH）** | 自身被消灭 | 文天祥、凤凰 |
| **受伤时（ON_DAMAGED）** | 自身受到任意伤害 | 项羽、科比 |
| **攻击时（ON_ATTACK）** | 自身发起攻击 | 德古拉（吸血） |
| **回合开始（START_OF_TURN）** | 己方回合开始 | 观音菩萨、太上老君 |
| **回合结束（END_OF_TURN）** | 己方回合结束 | 巴菲特·价值投资 |
| **打出法术时（ON_SPELL_PLAYED）** | 己方打出任意法术 | 诸葛亮、邓布利多 |
| **打出单位时（ON_UNIT_PLAYED）** | 己方打出任意单位 | 孔子、比尔盖茨 |
| **任意单位消灭时（ON_ANY_DEATH）** | 场上任意单位消灭 | 马尔斯·战争、灭霸 |
| **敌方单位消灭时（ON_ENEMY_DEATH）** | 敌方单位被消灭 | 怪兽巢穴环境 |
| **战斗开始前（BEFORE_COMBAT）** | 战斗阶段启动时 | 潜伏揭示触发 |
| **受到法术影响时（ON_SPELL_TARGET）** | 被法术指定 | 银火龙（反弹） |

## 附：设计确认记录

以下问题已由设计方确认（2026-03-29）：

| # | 问题 | 确认结论 |
|---|------|---------|
| 1 | 英雄初始血量是多少？ | ✅ **20点** |
| 2 | 高地面道路（道路0）的特殊规则是什么？ | ✅ **无特殊放置规则**；部分卡牌在高地打出时有额外效果（卡牌效果层实现，非道路系统层）|
| 3 | 每张非传奇卡牌在卡组里最多放几张？ | ✅ **最多4张** |
| 4 | 同道路有多个协作单位时，攻击顺序如何？ | ✅ **距离对方英雄越近的越先攻击**；协作最多 **2个** 单位 |
| 5 | 溢伤X：X是固定值还是溢出值？目标是英雄还是道路后方下一目标？ | ✅ **固定值X**，打到该道路后方的**下一目标**（有单位先打单位，否则打英雄）|
| 6 | 法免单位是否能被己方法术/超能力影响？ | ✅ **法免仅免疫法术卡和超能力卡**；单位能力/效果不受法免影响 |
| 7 | 单位击杀对面单位后，剩余伤害是否传到英雄？ | ✅ **不传递**。每个单位每次战斗只攻击一次；击杀对面单位后战斗在该道路结束，英雄不受额外伤害。**穿透（PIERCE）**和**溢伤（OVERSHOOT）**是唯一例外，可在击杀后对英雄造成额外伤害 |
