# 英雄候选列表

> 每位英雄拥有两个属性，只能使用该两个属性下的卡牌构建卡组。
> 本文档为候选草稿，可自由修改替换。

---

## 属性总览

### 人类方
| 属性 | 核心特质 |
|------|----------|
| 💪 肉体 | 身体极限、格斗、不屈战斗力 |
| 🧠 脑力 | 谋略、智识、控场预判 |
| 💰 财富 | 资源、权势、商业帝国 |
| 🔥 精神 | 意志、信念、逆境爆发 |
| ⚡ 超能 | 超自然身体能力、异能、果实/查克拉等 |

### 非人类方
| 属性 | 核心特质 |
|------|----------|
| ✨ 神 | 神明、天界存在、信仰之力 |
| 👻 鬼 | 亡灵、不死、幽灵、死亡之力 |
| 🐉 兽 | 怪物、原始生命力、变异巨兽 |
| 🤖 械 | 机械生命、人造存在、科技造物 |
| 🌌 异 | 异星、异维度、宇宙级存在 |

> ⚠️ 法术/异术类属性（咒术回战、哈利波特等）作为 DLC 保留，暂不纳入初始版本。
