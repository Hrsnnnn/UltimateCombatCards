# 卡牌设计 — 初始版本（20张）

---

## 一、属性映射表

> 我们的属性与 PVZH 属性的对应关系，用于指导卡牌机制设计方向。

### 人类方 → PVZH 植物方

| 我们的属性 | 对应 PVZH 属性 | 核心机制借鉴 |
|------------|----------------|--------------|
| 💪 **肉体** | Guardian 守卫 + Mega-Grow 超级生长 | 高生命值、装甲、强化攻击、Double Strike |
| 🧠 **脑力** | Smarty 聪慧 | 弹射（Bounce）、冻结（Freeze）、抽牌、控场 |
| 💰 **财富** | Solar 太阳 | 回合开始产出资源、治疗英雄、Team-Up 增益 |
| 🔥 **精神** | Kabloom 大爆炸 | 逆境触发、死亡爆发、牺牲换取强力效果 |
| ⚡ **超能** | Mega-Grow 超级生长 | Double Strike、额外攻击、召唤克隆、进化强化 |

### 非人类方 → PVZH 僵尸方

| 我们的属性 | 对应 PVZH 属性 | 核心机制借鉴 |
|------------|----------------|--------------|
| ✨ **神** | Brainy 聪慧 + Beastly 野蛮 | 全场效果、召唤、Untrickable、治疗 |
| 👻 **鬼** | Sneaky 狡猾 + Crazy 疯狂 | Gravestone 隐藏、Anti-Hero、Deadly、死亡复活 |
| 🐉 **兽** | Beastly 野蛮 | Hunt 追踪、Frenzy 狂暴、高攻、登场爆发伤害 |
| 🤖 **械** | Brainy 聪慧/Science | 科技联动、Hunt、死亡触发、全场增益 |
| 🌌 **异** | Crazy 疯狂 + Sneaky 狡猾 | 随机效果、Overshoot、Anti-Hero、Strikethrough |

## 三、设计总结

| 属性 | 已设计卡牌 | 核心机制方向 |
|------|------------|--------------|
| 💪 肉体 | 项羽、李小龙 | 高生命受伤强化、Double Strike 连击 |
| 🧠 脑力 | 诸葛亮、孙子 | 法术联动抽牌、弹射控场 |
| 💰 财富 | 马云、巴菲特 | 资源产出、伤害转治疗 |
| 🔥 精神 | 科比、岳飞 | 逆境爆发、死亡反击 |
| ⚡ 超能 | 路飞、鸣人 | Double Strike 进化、克隆召唤 |
| ✨ 神 | 宙斯、如来 | 全场清小兵、Untrickable 持续治疗 |
| 👻 鬼 | 巫妖王、杰森 | Gravestone 复活、Deadly 不死 |
| 🐉 兽 | 哥斯拉、异形 | 登场爆发、Hunt 繁殖 |
| 🤖 械 | 擎天柱、终结者 | 族群增益、Hunt 死亡残骸 |
| 🌌 异 | 灭霸、超人 | 随机消灭、Bullseye+Frenzy 全能攻击 |
