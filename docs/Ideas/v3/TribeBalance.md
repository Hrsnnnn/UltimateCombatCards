## 二、待确认设计决策

### 2.1 历史族拆分方案（讨论中）

**现状**：历史族实际上只收录中国历史人物（项羽、诸葛亮、岳飞等），但名称叫"历史"容易误导。西方/日本历史人物目前分散在勇者、战士族群中：
- 阿基里斯、赫拉克勒斯、奥德修斯 → 勇者
- 织田信长、本多忠胜、真田幸村 → 战士

**推荐方案：拆分为四个族群**

| 原族群 | 拆分后 | 收录范围 | 现有卡牌归属变化 |
|--------|--------|---------|----------------|
| 历史 | **华夏** | 中国历史人物：项羽、诸葛亮、岳飞、孔子、韩信等 | 保持不变 |
| 新增 | **西史** | 西方真实历史人物：凯撒、拿破仑、亚历山大等 | 从历史族分拆 |
| 新增 | **战国** | 日本战国时代人物：织田信长、真田幸村、宫本武藏等 | 从战士族迁移 |
| 新增 | **神话** | 各国神话英雄（非神明）：赫拉克勒斯、阿基里斯、奥德修斯等 | 从勇者族迁移 |

**族群边界说明**：
- **华夏**：中国有据可查的历史人物
- **西史**：西方有据可查的历史人物（不含神话）
- **战国**：专指日本战国时代（1467-1615），与其他日本历史人物区分
- **神话**：各文化中的神话英雄（人类英雄，非神明）；神明类归入非人类方的神属性
- 赫拉克勒斯、阿基里斯等希腊英雄归入**神话**，不归西史

**好处**：
- 华夏族张数从32降至约18-20，恢复合理水平
- 战国族、神话族从零开始，吸收现有分散卡牌后各有8-12张基础盘
- 西史族留待后续扩充（凯撒、拿破仑等角色尚未入卡）

**待确认**：
- [ ] 历史族正式改名为"华夏"
- [ ] 西史族正式成立（目前暂无卡牌，待扩充）
- [ ] 战国族正式成立，从战士族迁移：织田信长、真田幸村、本多忠胜
- [ ] 神话族正式成立，从勇者族迁移：赫拉克勒斯、阿基里斯、奥德修斯

### 2.2 非人类小族群处理方案（讨论中）

**科技族（5张）**：与机甲高度重叠，建议**合并入机甲族**
- 终结者、天网等本质都是机械生命，区别不大

**巨人族（4张）**：张数过少，不适合作为独立族群设计协同
- 建议作为**跨族群特殊单位**存在，类似通用牌，不强求族群联动
- 巨人单位改为注明所属原来的属性（如进击巨人系列归入兽属性，机械巨人归入械属性）

### 2.3 平衡原则（确认中）

- 每个族群建议维持 **10-20 张**卡牌，低于8张难以形成族群协同
- 扩充弱势族群优先通过**新增角色**实现，而非强行改变现有卡牌归属
- 削减过大族群的方式是**提高设计门槛**（新历史/忍者卡需要更独特的效果才能入选），而非删卡
